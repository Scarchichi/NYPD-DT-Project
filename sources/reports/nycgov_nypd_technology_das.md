# NYPD Technology Page - Domain Awareness System

## Summary

This official NYPD technology page describes the mobile version of the Domain Awareness System (DAS) as a crime-fighting and counterterrorism tool jointly developed by the NYPD and Microsoft.

Key points:
- DAS uses the largest networks of cameras, license plate readers, and radiological sensors in the world.
- The system is connected to handheld devices, tablets, and Automatic Vehicle Locator (AVL) GPS systems installed in more than 5,000 vehicles.
- Mobile DAS gives precincts real-time access to surveillance feeds and sensor data that were previously available only to headquarters.
- NYPD frames the system as critical for fighting crime and terrorism while improving officer situational awareness.
